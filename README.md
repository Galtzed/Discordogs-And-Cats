A Discord Bot that uses:
* [Dog Ceo Api](https://dog.ceo/api)
* [Cat As A Service](https://cataas.com)

## Dependencies
This Bot NEEDS [Node.js](https://nodejs.org/en/).

## Running
To run the bot you just need to run the `run.bat` file.
